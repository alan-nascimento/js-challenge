# JavaScript Challenge

## Description

A practical JavaScript challenge.

## Prerequisites

```
$ git >= v2
$ npm >= v6
$ nodejs >= v12
```

## Running unit testing

1. Open a terminal execute:

```zsh
$ npx jest
```

2. Run the following command to collect coverage report:

```zsh
$ npx jest --coverage
```
